# 🚀 ATS Resume Analyzer

An AI-powered ATS Resume Analyzer that compares resumes with job descriptions and provides an ATS score, keyword analysis, and personalized suggestions to improve resume quality.

## ✨ Features

- Upload PDF resumes
- Compare resume with job description
- Generate ATS compatibility score
- Identify missing keywords
- AI-powered improvement suggestions

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- Google Gemini API
